# AI Voice Assistant

A **web-based AI voice assistant** powered by **LLaMA-3.3-70BVersatile**, **Whisper**, and **Flask**. This assistant can transcribe speech in real time, generate intelligent context-aware responses, and execute voice commands accurately, all through a clean and responsive web interface.

---

## Features

- **Real-Time Speech Recognition**: Uses Whisper to transcribe voice input accurately and efficiently.
- **Context-Aware Responses**: Powered by LLaMA-3.3-70BVersatile to provide intelligent, natural-sounding responses.
- **Voice Command Execution**: Executes commands like opening apps, performing searches, or controlling system functions.
- **Responsive Web Interface**: Clean and interactive frontend built with HTML, CSS, and JavaScript.
- **Robust API Communication**: Fast and reliable communication between frontend and backend ensures smooth conversational flow.

---

## Tech Stack

- **Backend**: Flask  
- **AI Models**: LLaMA-3.3-70BVersatile (for response generation), Whisper (for speech-to-text)  
- **Frontend**: HTML, CSS, JavaScript  
- **Others**: RESTful APIs for communication between frontend and backend
